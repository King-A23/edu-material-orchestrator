# Fixture Policy

Only commit fixtures that are safe to redistribute.

Required metadata for every fixture:

- source
- license or permission basis
- generation method
- intended evaluation use

Preferred fixture strategy for this repository:

- self-authored synthetic PPTX and scanned-PDF inputs
- generated test fixtures created at runtime by the test suite

Do not commit classroom materials, exam papers, answer keys, or student data unless redistribution rights are explicit and documented.
